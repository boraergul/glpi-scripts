# GLPI Plugins Suite

Bu dizin, GLPI core fonksiyonlarını genişletmek için geliştirilen özel PHP eklentilerini (plugins) içerir.

## 📂 Eklentiler

| Eklenti | Açıklama |
|---|---|
| [`slareport/`](file:///d:/Google%20Drive/Projeler/Script/plugin/slareport) | SLA ihlallerini görselleştiren ve detaylı raporlayan GLPI 10/11 uyumlu eklenti. |

## 🛠️ Kurulum Genel Bakış

1. Eklenti dizinini GLPI kurulumunuzun `plugins/` klasörüne kopyalayın.
2. GLPI arayüzünden **Setup > Plugins** (Yapılandırma > Eklentiler) kısmına gidin.
3. Eklentiyi bulun, **Install** (Kur) ve ardından **Enable** (Aktif et) butonlarına tıklayın.

## 📝 Notlar

- Bu eklentiler PHP 8.1+ ve GLPI 10/11 sürümleriyle uyumlu olacak şekilde geliştirilmektedir.
- Her eklentinin kendi dizini altında daha detaylı bir `README.md` dosyası bulunmaktadır.
